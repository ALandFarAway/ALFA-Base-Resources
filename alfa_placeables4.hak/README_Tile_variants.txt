This is a set of tiles that are intended to expand upon the existing
standard tile sets that come with the commercial Neverwinter Nights 2
game. They include various stairways, door, wall, and floor variants
that provide more variety and flexibility for your modules. The tile
set that are expanded upon are: Standard Interior, Standard Castle,
Standard Mine. No variants of the Crypt tileset were made; instead,
it is recommended that you try the CryptSet mod.

Here is a brief description of the added tile variants in this set:
